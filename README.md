## Setup 
* `.env` containing: 
  * GATEWAY_PORT
  * API_PORT
  * AUTHENTICATION_PORT
  * WEB_PORT
  * GOOGLE_APPLICATION_CREDENTIALS=/paperpod/service-account.json (if located in root)
* `docker-compose up -d`
